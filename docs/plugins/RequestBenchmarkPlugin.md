# RequestBenchmarkPlugin  

### Example `appsettings.json`  
```json
{
  // Determines whether or not to include gRPC requests in benchmarks
  "IgnoreGrpcRequests": false,
  "Logging": {
    "LogLevel": {
      "Default": "Information",
      "Microsoft.AspNetCore": "Warning"
    }
  },
  "AllowedHosts": "*"
}
```
